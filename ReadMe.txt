Email Login, Upload attachment and send testing with Selenium

Upload attachment with AutoIT tool and Extent Report will be creat after testing

Need to set your own path in ExtentFactory.java and ScreenShot.java for ExtentReport and screenshot photo.

And also need to set your own attachment path in \TestNGEmailLogin\script\fileUpload.au3

Run testSuite.xml then 2 test class will running in one time
